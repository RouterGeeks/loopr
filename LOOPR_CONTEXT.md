# Loopr Context

Loopr is a mobile-first PWA for capturing open loops and deciding what to do with them.

Core flow:
Capture → Do / Delay / Drop → Revisit

Core rule:
Frictionless capture is the #1 priority.

If capture takes more than 2–3 seconds, the app is broken.

V1 features:
- Quick text capture
- Voice capture later
- Do / Delay / Drop
- Delay with reminder date/time
- Revisit view
- Local storage / IndexedDB
- No login
- No backend
- No AI processing in V1

Tone:
Calm, supportive, low-friction.

Design:
Retro motel keychain feel.
Muted lavender, seafoam, cream, charcoal.

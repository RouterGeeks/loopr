# Loopr Project Plan

## V1 Goal
Build a mobile-first PWA that lets users capture thoughts quickly, decide Do / Delay / Drop, and revisit delayed loops.

## Tech Stack
- React
- Vite
- TypeScript
- Tailwind CSS
- IndexedDB or localStorage
- PWA support

## Sprint 1: App Shell
- Create Vite React TypeScript app
- Add Tailwind
- Create mobile-first layout
- Add navigation: Home, Revisit, Settings

## Sprint 2: Capture
- Add quick capture input
- Save loop locally
- Auto-focus capture field
- Show captured loops immediately

## Sprint 3: Do / Delay / Drop
- Add status buttons
- Do sets status to do
- Delay opens date/time picker
- Drop archives or hides item

## Sprint 4: Revisit
- Show delayed loops where reminder time has passed
- Allow user to Do, Delay again, or Drop

## Sprint 5: Polish
- Add search
- Add delete/export data
- Add installable PWA support
- Add privacy statement

## Not V1
- No login
- No cloud sync
- No GitHub integration
- No Obsidian integration
- No AI organization
